# Delivery Service System

Папка содержит аналитические материалы по системе обработки и доставки заказов.

Проект описывает IT-систему для приёма заказов, их распределения между курьерами, контроля доставки, передачи данных в бухгалтерию и управления доступами пользователей.

## Описание предметной области

Система предназначена для автоматизации работы сервиса доставки. Операторы вносят заказы в систему, курьеры выбирают и выполняют доставки, диспетчер контролирует процесс, а бухгалтерия получает данные для расчётов с поставщиками и курьерами.

В предметной области рассматриваются роли оператора, курьера, диспетчера, администратора, бухгалтерии, поставщиков заказов и внешних IT-систем.

## Основные процессы

В рамках проекта прорабатываются процессы, связанные с:

- получением и регистрацией заказов;
- вводом заказов оператором;
- распределением заказов между курьерами;
- выбором и бронированием заказа курьером;
- получением заказа в точке забора;
- доставкой заказа клиенту;
- контролем доставки диспетчером;
- переназначением заказов;
- передачей данных в бухгалтерскую систему;
- расчётом оплаты курьерам;
- регистрацией курьеров и управлением правами доступа.

## Цель проекта

Цель проекта — показать анализ системы доставки заказов: от выявления участников процесса и их проблем до описания требований, потоков данных, процессов и информационной модели.

Проект демонстрирует навыки анализа предметной области, формализации требований, моделирования процессов и подготовки аналитических таблиц и диаграмм.
