# User Stories

Раздел содержит пользовательские истории для системы онлайн-записи в барбершоп.

User Stories описывают потребности пользователей, ожидаемую ценность, критерии приёмки и тестовые сценарии.

## Состав раздела

- пользовательские истории для менеджера;
- пользовательские истории для клиента и мастера;
- критерии приёмки;
- тестовые сценарии;
- результаты обсуждения и согласования пользовательских историй.
