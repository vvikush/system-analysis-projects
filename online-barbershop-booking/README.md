# Online Barbershop Booking System

Папка содержит аналитические материалы по системе онлайн-записи клиентов на услуги барбершопа.

Проект описывает IT-систему, которая помогает клиентам записываться на услуги, а менеджерам и мастерам — работать с расписанием, записями, услугами, отзывами и связанными бизнес-процессами.

## Описание предметной области

Система предназначена для автоматизации процесса записи клиентов в барбершоп. Клиент может выбрать услугу, мастера и доступное время, а сотрудники барбершопа получают инструменты для управления расписанием, услугами, оплатой и обратной связью.

В предметной области рассматриваются роли клиента, посетителя, мастера, менеджера, администратора, руководителя сети, внешнего регулятора и команды разработки.

## Основные процессы

В рамках проекта прорабатываются процессы, связанные с:

- онлайн-записью клиента;
- выбором услуги, мастера и времени;
- управлением расписанием мастеров;
- отправкой напоминаний клиентам;
- обработкой отзывов;
- фиксацией оказанных услуг;
- оплатой и передачей данных в бухгалтерию;
- просмотром расписания и записей мастером;
- формированием отчётов;
- электронным учётом санитарной обработки.
