# Diagrams

Папка содержит диаграммы и визуальные модели по системе онлайн-записи клиентов на услуги барбершопа.

## Состав диаграмм

| Файл | Диаграмма | Назначение |
|---|---|---|
| [er-diagram.png](./er-diagram.png) | ER Diagram | Показывает основные сущности модели данных и связи между ними. |
| [data-flow-diagram.png](./data-flow-diagram.png) | Data Flow Diagram | Показывает потоки данных между системой и участниками процесса. |
| [swimlane-diagram.png](./swimlane-diagram.png) | Swimlane Diagram | Показывает распределение ответственности между ролями в процессе. |
| [additional-swimlane-diagram.png](./additional-swimlane-diagram.png) | Additional Swimlane Diagram | Показывает процесс с создаваемыми и обновляемыми артефактами. |
| [state-diagram.png](./state-diagram.png) | State Diagram | Показывает жизненный цикл визита клиента. |
| [process-bpmn-booking.png](./process-bpmn-booking.png) | BPMN Diagram | Показывает ключевой бизнес-процесс оформления онлайн-записи клиента. |

## Назначение

Диаграммы дополняют текстовые аналитические материалы и помогают быстро понять:

- какие сущности лежат в основе модели данных;
- какие данные передаются между участниками процесса;
- как распределена ответственность между ролями;
- какие артефакты создаются и обновляются в процессе;
- как меняется состояние визита клиента;
- как выполняется ключевой процесс онлайн-записи.
