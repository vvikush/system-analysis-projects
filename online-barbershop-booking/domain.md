# Domain

## Описание предметной области

Предметная область проекта — онлайн-запись клиентов на услуги сети барбершопов.

Система BARB рассматривается как решение для записи клиентов на услуги, выбора мастера и времени, управления расписанием мастеров, обработки записей, отправки напоминаний, фиксации оказанных услуг, сбора отзывов и передачи данных для отчётности.

В рамках предметной области рассматриваются как клиенты и сотрудники барбершопа, так и внешние участники, связанные с отчётностью, санитарными требованиями и сопровождением системы.

## Участники предметной области

### Зарегистрированный посетитель / клиент

Клиент использует систему для записи на услуги, получения напоминаний, просмотра скидок, истории посещений и будущих записей.

### Незарегистрированный посетитель

Незарегистрированный посетитель может просматривать услуги и расписание мастеров, пройти саморегистрацию или выполнить быструю запись с минимальным набором персональных данных.

### Мастер

Мастер оказывает услуги клиентам и использует систему для просмотра своего расписания, занятости и информации о записанных клиентах.

### Менеджер

Менеджер управляет расписанием мастеров, услугами, записями клиентов, шаблонами сообщений, отчётами, отзывами и электронным учётом санитарной обработки.

### Администратор системы

Администратор управляет пользователями, ролями, правами доступа, журналом аудита и контролирует стабильность системы.

### Руководитель сети

Руководитель получает отчёты и статистику, анализирует клиентскую базу, загрузку мастеров и эффективность работы барбершопа.

### Санэпиднадзор

Санэпиднадзор является внешним регулятором, связанным с требованиями к санитарной обработке и отчётности.

### Регулятор / ФНС

Регулятор является внешней стороной, связанной с требованиями к фискализации, отчётности и данным о платежах при наличии соответствующей интеграции.

### Команда разработки

Команда разработки собирает и уточняет требования, разрабатывает, тестирует, сопровождает и дорабатывает систему.

## Основные объекты предметной области

| Объект | Описание |
|---|---|
| Клиент | Зарегистрированный посетитель, который использует систему для записи, получения скидок и напоминаний. |
| Незарегистрированный посетитель | Пользователь, который может просматривать услуги и расписание, а также записаться или пройти саморегистрацию. |
| Мастер | Сотрудник барбершопа, оказывающий услуги клиентам. |
| Менеджер | Сотрудник, управляющий расписанием, услугами, записями и шаблонами сообщений. |
| Администратор системы | Пользователь, управляющий ролями, правами доступа и технической поддержкой системы. |
| Услуга | Действие или работа, оказываемая клиенту в барбершопе. |
| Тип услуги | Категория услуги, используемая для систематизации услуг. |
| Расписание | График работы мастеров и доступных временных интервалов для записи. |
| Онлайн-запись | Самостоятельный выбор услуги, мастера, даты и времени через систему. |
| Напоминание | Автоматическое сообщение клиенту о предстоящей записи. |
| Канал связи | Способ передачи сообщения клиенту, например Telegram, WhatsApp, VK или СМС. |
| Оплата | Процесс приёма денежных средств за оказанную услугу и передачи данных для учёта. |
| Отзыв | Оценка и комментарий клиента после посещения услуги. |
| Санитарная обработка | Обработка помещения по санитарным требованиям. |

## Основные процессы предметной области

В рамках предметной области рассматриваются следующие процессы:

- просмотр услуг;
- просмотр расписания мастеров;
- онлайн-запись клиента на услугу;
- запись без обязательной регистрации;
- регистрация посетителя;
- перенос записи;
- отправка напоминаний клиенту;
- управление расписанием мастеров;
- управление списком услуг;
- оказание услуги клиенту;
- фиксация результата оказанной услуги;
- приём оплаты;
- передача данных об оплате в бухгалтерию;
- сбор отзывов клиентов;
- формирование отчётов;
- ведение электронного учёта санитарной обработки;
- управление пользователями, ролями и правами доступа.
