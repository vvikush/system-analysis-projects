# Use Cases

Раздел содержит варианты использования системы онлайн-записи в барбершоп.

Use Cases описывают подробное взаимодействие акторов с системой: цели, предусловия, триггеры, постусловия, основной сценарий, альтернативные сценарии, исключительные потоки, а также связи include и extend.

## Состав раздела

- управление актуальным расписанием услуг мастеров;
- выбор услуги, мастера и слота клиентом;
- бронирование услуги клиентом;
- просмотр мастером расписания, записей и отзывов.
